# Python Fundamentals Samples

A collection of Python scripts demonstrating fundamental algorithms, data structures, and common coding interview patterns.

## Contents

| Script | Description |
| :--- | :--- |
| `LRUCache.py` | Implementation of a Least Recently Used (LRU) Cache with O(1) time complexity for get and put operations. |
| `anagrams.py` | A function to group a list of strings into groups of anagrams. |
| `hashtables.py` | Demonstrates core dictionary operations, `defaultdict` usage, and the classic "Two Sum" problem implementation. |
| `longestconsecutive.py` | Finds the length of the longest consecutive elements sequence in an unsorted array. |
| `sorting.py` | Implementations of Quicksort, Mergesort, and a color sorting algorithm (Dutch National Flag problem). |
| `subarraySum.py` | Calculates the total number of continuous subarrays whose sum equals a given target `k` using a hash map. |
| `wordPattern.py` | Determines if a string follows a specific pattern (e.g., "abba" matches "dog cat cat dog"). |

## Prerequisites

- Python 3.x

## How to Run

To run any of the scripts and see their output (provided via test cases at the end of each file), use the Python interpreter:

```bash
python3 <script_name>.py
```

Example:
```bash
python3 sorting.py
```
